==>  Project Overview

The AI Garbage Analyzer is a smart system designed to identify and classify garbage using Artificial Intelligence.
The goal of this project is to promote waste management and environmental awareness by automating garbage detection.

🛠️ Technologies Used
. Python
. YOLOv8 (Object Detection Model)
. HTML, CSS (Frontend)
. Flask (Backend)

📂 Project Structure
├── app.py              # Main backend application
├── index.html          # Frontend UI
├── style.css           # Styling file
├── yolov8.pt           # Trained model
├── dataset/            # Dataset (if included)
├── static/             # Static files (CSS, images)
└── templates/          # HTML templates
